# UK Road Accident Analysis

## Overview
Data analysis of the UK Department for Transport 2019 road accident dataset focusing on the Hull/Yorkshire region.

The project analyzes when and under what conditions road accidents occur and identifies risk patterns that can inform road safety interventions.

## Key Analysis
- Peak accident hours
- Pedestrian accident patterns
- Motorcycle accident severity by engine size
- Weekend vs weekday accident trends
- Time-series accident forecasting

## Key Findings
- Accident frequency peaks around **17:00–18:00**, about **18.5% higher than the daily average**, indicating strong commuter risk patterns.
- Pedestrian accidents frequently occur during evening hours and clear weather conditions.
- Larger motorcycle engine categories show higher severe injury rates.

## Methods
- Exploratory Data Analysis
- Statistical testing
- Time-series modelling
- Logistic regression
- Risk factor analysis

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn
